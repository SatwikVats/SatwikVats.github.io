Key-pair

Creating key-pair using AWS CLI. #Here, we may modify the arguments including key-pair names. aws ec2 create-key-pair --key-name FirstKeyPair --query 'KeyMaterial' --output text > FirstKeyPair.pem To check whether key-pair has been made correctly, we can either visit the console manually or use the following command. aws ec2 describe-key-pairs Listing down all the key-pairs created by user. This generates all the key-pairs. Troubleshooting : If you are unable to generate list of Key-pairs due to output error, try changing the output type to json in configuration.

Vpc

Created a VPC(Virtual Private Cloud) for my infrastructure. VPC is an isolated network in your AWS environment. Copied the appropriate resources tags from Terraform official documentation into the main.tf file. ID of subnet should be defined according to the name of corresponding vpc. resource "aws_vpc" "firstvpc" { cidr_block = "10.0.0.0/16" tags = { Name = "vpc1" } } resource "aws_subnet" "subnet1" { vpc_id = aws_vpc.firstvpc.id cidr_block = "10.0.1.0/24" tags = { Name = "Main" } }
